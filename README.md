# IPL Data Analysis (2008–2023)

## Project Objective
Analyze IPL match data to find trends in teams, stadiums, and scoring patterns.

## Dataset
Source: Kaggle IPL Dataset (2008–2023)

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Questions
-Which team wins the most matches?
-Which stadium has the highest scoring games?
-Who are the top batsmen?
-Toss impact on match results.

## Visualizations
- Most of the match winner teams
- Top 10 batsman
- Toss winner vs Match winner

## Key Insights
- M. Chinnaswamy Stadium has some of the highest scoring matches.
- Certain teams perform better when batting first.
